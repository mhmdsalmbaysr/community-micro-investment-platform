# 🌐 منصة الاستثمار المجتمعي | Community Micro-Investment Platform

> **MVP / Demo / Proof of Concept** — منصة ويب تجريبية لدعم المشاريع الصغيرة والفئات الهشة عبر الاستثمار المجتمعي.
> ⚠️ هذا النظام **محاكاة فقط** ولا يتضمن أي تحويلات مالية حقيقية.

## 🎯 الهدف
عرض الفكرة على الجهات المانحة وإثبات نموذج العمل والأثر الاجتماعي، واختبار تفاعل المستخدمين قبل التطوير الكامل.

## ✨ المكوّنات
- **Landing Page** احترافية (Hero / المشكلة / الحل / كيف تعمل / الأثر / الشركاء / CTA).
- **صفحة المشاريع** مع فلاتر وحالات (مفتوح / قيد التنفيذ / مكتمل) وأشرطة تقدّم.
- **تفاصيل المشروع** + **استثمار تجريبي** (يُحفظ في المتصفح).
- **لوحة المستثمر** (محفظة، عائد افتراضي، سجل، إشعارات).
- **لوحة صاحب المشروع** (إضافة مشروع، متابعة التمويل والمراجعة).
- **لوحة الإدارة** (قبول/رفض، إدارة المستخدمين، إحصائيات).
- **API تجريبي** (auth / projects / invest / admin).

## 🛠️ التقنيات
- [Next.js 14](https://nextjs.org) (App Router)
- [TailwindCSS](https://tailwindcss.com)
- بيانات تجريبية (Seed Data) + محاكاة الاستثمار عبر `localStorage`

## 🎨 الهوية البصرية
- أزرق = ثقة • أخضر = نمو • تصميم SaaS حديث • Responsive بالكامل • RTL.

## 🚀 التشغيل محلياً
```bash
npm install
npm run dev
# افتح http://localhost:3000
```

## 📁 الهيكل
```
app/
  page.js                 # Landing
  projects/               # قائمة وتفاصيل المشاريع
  dashboard/investor/     # لوحة المستثمر
  dashboard/owner/        # لوحة صاحب المشروع
  admin/                  # لوحة الإدارة
  api/                    # مسارات API التجريبية
components/               # مكوّنات واجهة قابلة لإعادة الاستخدام
lib/                      # بيانات تجريبية + متجر العميل
```

## 🔑 تسجيل الدخول التجريبي
صفحة `/login` تتيح الدخول مباشرة باختيار الدور (مستثمر / صاحب مشروع / إدارة).

## 🗺️ خارطة الطريق
- مصادقة حقيقية + قاعدة بيانات.
- بوابة دفع/محفظة فعلية.
- تطبيق موبايل.
